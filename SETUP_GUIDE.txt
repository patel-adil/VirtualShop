VIRTUALSHOP - PREMIUM MARKETPLACE SETUP
============================================

Welcome to VirtualShop - Your Premium Shopping Experience!

Follow these steps to get your marketplace running:

STEP 1: Install Dependencies
-----------------------------
Open PowerShell in this folder and run:
    npm install

This will install all required packages:
  - React 18.2.0
  - React Router DOM 6.20.0
  - Firebase 10.7.0
  - React Scripts 5.0.1


STEP 2: Configure Firebase Authentication
------------------------------------------
1. Go to https://console.firebase.google.com/
2. Click "Add project" or select an existing project
3. Enter a project name (e.g., "VirtualShop") and follow the setup wizard
4. Once created, go to "Authentication" from the left menu
5. Click "Get Started" if it's your first time
6. Click on "Sign-in method" tab
7. Click on "Google" provider
8. Toggle "Enable" and click "Save"

9. Now get your configuration:
   - Click the gear icon next to "Project Overview"
   - Select "Project settings"
   - Scroll down to "Your apps"
   - Click the web icon (</>) to add a web app
   - Give it a nickname (e.g., "VirtualShop Web")
   - Click "Register app"
   - Copy the firebaseConfig object

10. Open src/firebase.js in this project
11. Replace the placeholder values with your actual Firebase config:
    
    const firebaseConfig = {
      apiKey: "YOUR_ACTUAL_API_KEY",
      authDomain: "YOUR_PROJECT.firebaseapp.com",
      projectId: "YOUR_PROJECT_ID",
      storageBucket: "YOUR_PROJECT.appspot.com",
      messagingSenderId: "YOUR_SENDER_ID",
      appId: "YOUR_APP_ID"
    };


STEP 3: Start the Development Server
-------------------------------------
In PowerShell, run:
    npm start

The app will automatically open in your browser at:
    http://localhost:3000

You should see the beautiful VirtualShop interface!

TROUBLESHOOTING
---------------

Problem: "npm is not recognized"
Solution: Install Node.js from https://nodejs.org/ (Download LTS version)

Problem: Firebase authentication not working
Solution: 
  - Ensure you updated src/firebase.js with YOUR actual config
  - Make sure Google sign-in is enabled in Firebase Console
  - Check browser console (F12) for specific error messages
  - Verify your domain is authorized in Firebase settings

Problem: Styles not loading properly
Solution:
  - Clear browser cache (Ctrl + Shift + Delete)
  - Hard refresh the page (Ctrl + F5)
  - Check if Poppins font is loading (F12 > Network tab)

Problem: Port 3000 already in use
Solution: 
  - The app will prompt to use another port, press 'Y'
  - Or manually specify port: set PORT=3001 && npm start

Problem: Build errors
Solution: 
  - Delete node_modules folder
  - Delete package-lock.json
  - Run: npm cache clean --force
  - Run: npm install
  - Run: npm start
